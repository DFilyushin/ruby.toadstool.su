История изменений

Версия 19.12
  • Быстрое создание списка квартир
  • Добавлена проверка файла оплат при импорте (выводится информация количестве записей, общей сумме оплаты, сообщение о несовпадении сервисного кода файла и кода, установленного в системе, а также предупреждение о различных датах в файле)
  • Исправлены мелкие ошибки с сохранением настроек
Версия 19.11
  • печать квитанций возможно только на лицевые счета с долгом
  • оборотная ведомость по домам печатается в «старом стиле» по умолчанию
  • мелкие доработки интерфейса по пожеланиям пользователей (уменьшены размеры окон, не вмещались на экран)
  • добавлена сортировка в некоторые журналы (сортировка при нажатии на заголовок журнала)
Версия 19.10
  • Добавлен новый вид основного начисления суммой
  • Исправлены ошибки импорта оплат и несколько ошибок интерфейса в журнале платежей
Версия 19.03
  • Добавлена возможность выбора нескольких файлов оплат от Теплоцентра
  • Добавлена возможность пересканирования таблицы соответствия для поиска добавленных квартир в жилой фонд, но недобавленных в список.
  • Добавлена возможность выбора поля импорта даты для файлов оплат Теплоцентра. Доступны для выбора поля: дата фактической оплаты и дата проведения
    • документов оплаты. Обычно, даты совпадают, но в конце и начале месяца даты могут несовпадать и суммы оплат будут «теряться», переходить в другой месяц.
Версия 19.01
  • Добавлена возможность удаления пачек из оплат
  • Отчёт «Оборотно-сальдовая ведомость по КСК» переименован в «Оборотно-сальдовая ведомость по предприятию»
  • В отчёт «Оборотно-сальдовая ведомость по предприятию» в заголовок выводится наименование предприятия
  • Отчёты, содержащие в название «КСК» переименованы как «предприятие»
Версия 18.12
  • Добавлен модуль ввода первичных данных через файл Excel. Шаблон файла (Шаблон импорта данных.xltx) расположен в папке templates.
Версия 18.11
  • в журнале ввода остатков исправлен вывод номера лиц.счёта
  • в журнале ввода остатков исправлен выбор дома, отображаются только нужные реквизиты
Версия 18.10
  • мелкие исправления в каталоге домов и квартир (реакция на ентер при редактировании)
  • исправлено формирование наименования дома. Теперь наименование формируется из указанного адреса (улицы и номер дома)
  • добавлена возможность изменения справочника улиц города. При редактировании быть внимательным. Справочник ведётся в Теплоцентре, коды должны соответствовать кодам справочника Теплоцентра.
Версия 17.11
  • Добавлен отчёт Оборотная ведомость по ФР по КСК. Собирает данные по общему начислению и оплатам на определённую дату. Выводит итог
Версия 16.11
  • добавлена возможность работы с ПавлодарЭнергоСервис (экспорт оборотов, импорт платежей)
  • исправлен отчёт в отдел статистики
Версия 16.9
  • Признак старшего по домам используется при начислениях. Для этого добавлена опция в справочник видов начислений «Учитывать распределения начисления по старшему по дому». Процедура начисления сначала выясняет алгоритм начисления на старшего из справочника домов. В случае, когда такой признак установлен, начисление производится с учётом флага.
  • Добавлен новый отчёт «Реестр счёт-квитанций» для удобного формирования ежедневного реестра полученных оплат
Версия 15.01.23
  • Добавлена возможность работы с фондом развития жилого массива
  • в карточку дома добавлена настройка «Фонд развития», отметка которой позволяет вести расчёт по фонду развития для выбранного дома
  • добавлена настройка процента для расчёта тарифа по фонду развития
  • изменён принцип формирования старого вида отчёта Оборотная ведомость по домам, формируем на всех (без учёта физ\юр. лица), а также площадь берём из начисления
  • исправлена старая ошибка, требующая перезапуска программы чтобы данные из справочников попали в выбор
  • общая оптимизация кода
Версия 14.01
  • Введена новая нумерация версий программы. Первая цифра указывает год, вторая месяц. По версии всегда можно узнать когда у вас в последний раз была обновлена программа.
  • Исправлена ведомость оплат, не добавляются лишние записи
  • Отчёты сжимаются по центру и не выходят за рамки

  • Добавлено окно просмотра итогов по пачкам в экран ввода оплат
Версия 5.5.1 (Март 2013)
